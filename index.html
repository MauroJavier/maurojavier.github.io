on Sat, Apr 09, 2016 by Mauro Javier · 1 min reading time

C:\data\db> 3.- Para que los comandos de consola funcionen tienes que agregar la ruta bin a tus variables de entorno, usualmente es C:\Program Files\MongoDB\Server\3.0\bin 4.- Prueba tu instalaci√≥n :) abre la terminal y ejecuta el comando ‘mongod’ para iniciar el servicio y en otra terminal ejecuta el comando ‘mongo’ para iniciar el cliente.
