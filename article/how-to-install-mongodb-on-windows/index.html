how to install mongodb on windows

on Sun, Mar 13, 2016

Sometimes you need mongodb installed locally for development purposes, and here is how to install mongodb on your windows machine.

1.- First go to the site https://www.mongodb.org/ and download the .msi package

2.- After that create the next folder structure in your hard drive.

C:\data\db>

3.- Now you need to set your environment variables.

Add the bin path to the mongod and mongo executables to your Path environment variable usually C:\Program Files\MongoDB\Server\3.0\bin

4.- Try your installation :)

open the cmd terminal and type mongod to start the service and in another terminal execute the mongo client
